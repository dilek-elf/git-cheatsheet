# Git & GitHub Survival Guide 🧠

A visual cheatsheet for the most important Git commands —
built as a dark-themed HTML page with neon styling.

## What it covers
- First-time Git setup
- Daily workflow (add, commit, push)
- GitHub & remote repositories
- Safe undo commands
- Branching basics

## Built with
HTML | CSS

## Why I built this
To practice HTML and CSS while creating something I actually use —
a reference page I can open anytime I forget a Git command.
